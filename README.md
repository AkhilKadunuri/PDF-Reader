# 📄 PDF to Speech Converter

A simple Python application that converts PDF text into speech using text-to-speech technology.

## 🚀 Features
- 📂 Select any PDF file
- 📖 Choose how many pages to read
- 🔊 Converts text into audio speech
- 🖥 Simple and beginner-friendly implementation

## 🛠 Technologies Used
- Python
- pyttsx3 (Text-to-Speech Engine)
- PyPDF2 (PDF Reader)
- Tkinter (File Dialog GUI)

## 📦 Installation

Install dependencies:

pip install -r requirements.txt

## ▶️ How to Run

python pdf_to_speech.py
